### :apple: Interval Tree

- 여러 폐구간이 있을 때 겹치는 경우를 계산하는 트리
- 폐구간의 insert, delete, search
- search는 겹치는 구간을 계산

- [a , b] , [c ,d] 두 구간이 겹친다는 것은
  a <= d && c <= b 이여야 한다.

- interval tree 는 균형트리고 트리를 구성하는 키는 구간의 첫번째 값이다.

```
[0, 3]
[5, 8]
[6, 10]
[8, 9]
[15, 23]
[16, 21]
[17, 19]
[19, 20]
[25 30]
[26, 26]

```
